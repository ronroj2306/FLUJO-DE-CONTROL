# Actividad 2 - Flujo de Control Parte 1

## Descripción

En esta actividad se desarrolló un programa en C++ que calcula el Impuesto Sobre la Renta (ISR) de un empleado a partir de su sueldo mensual.

El programa solicita el sueldo mensual, calcula el ingreso anual y, con base en la escala salarial establecida por la DGII, determina si el empleado debe pagar ISR. Si corresponde, muestra el monto del impuesto mensual y el sueldo neto. En caso de que el empleado esté exento, el programa indica que el ISR es **0 (No aplica)**.
